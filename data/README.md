# Data Directory Template

This directory keeps only the expected folder structure for local datasets.

Actual dataset files, generated tiles, and split text files are ignored by Git.

Main dataset roots used by this repository:

- `DynamicEarthNet/preprocessed/tiles512`
- `HRSCD/preprocessed/tiles500`
- `HRSCD/preprocessed/tiles2000`
- `lsmd_5186_final/tiles512_posaware_b2`
- `lsmd_5186_final/tiles512_posaware_b2_synth_v1`
- `re_lsmd/tiles512_posaware_b2`
- `re_lsmd/tiles512_posaware_b2_synth_v1`

For detailed expected layouts, see the top-level `README.md`.
